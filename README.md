# GIF → Video → Optimized Lottie

This repository hosts a simple web app by **Abdul Rehman Khan** that converts animated GIFs into transparent WebM videos and optimized Lottie JSON animations. The app is live via GitHub Pages.

## Live Demo

The app is served directly from GitHub Pages at:

```
https://<YOUR_GITHUB_USERNAME>.github.io/<REPO_NAME>/
```

Replace `<YOUR_GITHUB_USERNAME>` and `<REPO_NAME>` with your details.

## Files to Upload

* **index.html** — the main web app file.
* **favicon.png** — your profile icon (48×48 PNG) named `favicon.png`.
* (Optional) **README.md** — this file.

Your repository structure should look like:

```
/ (root)
├── index.html
├── favicon.png
└── README.md
```

## Enabling GitHub Pages

1. Commit and push these files to the `main` branch of your repository.
2. In your repository settings on GitHub, go to **Pages** (formerly GitHub Pages).
3. Under **Source**, select `main` branch and `/ (root)` folder.
4. Click **Save**.
5. After a minute, your site will be available at `https://<YOUR_GITHUB_USERNAME>.github.io/<REPO_NAME>/`.

## Updating the App

* To change the profile icon, replace `favicon.png` and commit.
* To update the app logic or UI, modify `index.html`.
* Push changes to `main` to automatically update the live site.
